## 1.1.0 - Released (Trillium R2 2025)
This release focused on upgrading to Spring Boot 4.0 / Spring 7.0, enhancements for Mosaic integration, and adding the "Generate Integration Templates" endpoint.

[Full Changelog](https://github.com/folio-org/mod-mosaic/compare/v1.0.0...v1.1.0)

### Stories
* [MODMO-28](https://folio-org.atlassian.net/browse/MODMO-28) - Remove account for the Mosaic organization
* [MODMO-27](https://folio-org.atlassian.net/browse/MODMO-27) - Implement endpoint for "Generate Integration Templates"
* [MODMO-26](https://folio-org.atlassian.net/browse/MODMO-26) - Change scope of spring-boot-starter-test from compile to test
* [MODMO-25](https://folio-org.atlassian.net/browse/MODMO-25) - mod-mosaic: Use GitHub Workflows for Maven
* [MODMO-22](https://folio-org.atlassian.net/browse/MODMO-22) - Upgrade module to Spring Boot 4.0 and Spring 7.0
* [MODMO-21](https://folio-org.atlassian.net/browse/MODMO-21) - Enhancements for Mosaic integration
* [MODMO-19](https://folio-org.atlassian.net/browse/MODMO-19) - Add reference template for Mosaic integration and default organization during startup

### Dependencies
* Bump `spring-boot` to `4.0.x`
